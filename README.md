# Sistema-de-Gerenciamento-Acad-mico
Este projeto implementa um sistema simples e modular para **gerenciamento acadêmico de estudantes**.   Ele permite cadastrar alunos, registrar notas, calcular médias e verificar situações de aprovação ou reprovação.   O código segue as diretrizes da **PEP 8**, garantindo clareza, padronização e documentação interna por meio de docstrings.
