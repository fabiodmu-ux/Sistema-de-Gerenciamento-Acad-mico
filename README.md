# 📘 Sistema de Gerenciamento Acadêmico

## 🔎 Introdução
Este projeto implementa um sistema simples e modular para **gerenciamento acadêmico de estudantes**.  
Ele permite cadastrar alunos, registrar notas, calcular médias e verificar situações de aprovação ou reprovação.  
O código segue as diretrizes da **PEP 8**, garantindo clareza, padronização e documentação interna por meio de docstrings.

---

## ⚙️ Pré-requisitos
Antes de executar o sistema, certifique-se de que possui:
- **Python 3.11+** instalado em sua máquina.
- Conhecimento básico de execução de scripts via terminal.
- Editor de texto ou IDE (VS Code, PyCharm, etc.) para visualizar e editar o código.
